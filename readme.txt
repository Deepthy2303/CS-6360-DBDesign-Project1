1. Extract the .zip file to a desired location.
2. Start the MySQL server.
3. Open the command terminal where the DBScript.bash script is located.
4. In the command terminal, type the following : sh DBScript.bash
5. Type	 javac src/com/dbproject/lms/*.java
6. Type	 java src.com.dbproject.lms.SearchGUI
7. The application has been launched.

The only technical dependency is that the system must support JAVA and MYSQL. No other extra libraries are required.
